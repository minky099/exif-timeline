MP4 촬영시각 최소 검증 MVHD

동영상 1개의 MP4 컨테이너 creation_time(mvhd·tkhd·mdhd)을 임의 시각으로 바이너리 패치해 저장 → iOS 사진앱이 그 시각을 촬영일로 인식하는지 실기기에서 확인하는 페이지입니다. 재인코딩 없이 몇 바이트만 바꿉니다.

1 MP4 1개 선택
2 기입할 촬영시각
MP4 박스 mvhd · tkhd · mdhd의 creation_time/modification_time에 이 시각(UTC 변환)을 씁니다.
3 기입 & 저장
✓ 검증 절차
  1. 위에서 저장(공유시트 비디오 저장 또는 다운로드 → 파일앱 → 사진앱 가져오기)
  2. 사진앱에서 방금 저장된 동영상의 정보(ⓘ) 확인 → 날짜/시각이 기입값과 같은지
  3. 일치하면 검증 성공 → 본 앱에 동영상 지원을 통합할 수 있음
  4. 다르면(저장 시각으로 잡힘) → 저장 경로를 바꿔 재시도, 결과를 공유해 주세요
